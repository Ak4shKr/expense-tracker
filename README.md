## ğŸš€ Getting Started
  ```bash
   git clone https://github.com/ak4shKr/expense-tracker
   ```


### Backend Setup

1. **Install dependencies:**
   ```bash
   cd backend
   npm install
   ```

2. **Set up `.env` file:**
   ```
   DB_URL=your_mongodb_connection_string
   JWT_SECRET=your_secret_key
   PORT=3000
   ```

3. **Run the server:**
   ```bash
     npm start
   ```
---

### Frontend Setup

1. **Navigate to frontend directory and install dependencies:**
   ```bash
   cd frontend
   npm install
   ```

2. **Run the frontend:**
   ```bash
   npm run dev
   ```
